Project: Educational Data Mining & Performance Prediction
Overview: > This project applies Machine Learning to predict student outcomes based on socio-economic and academic features.
The "Academic" Edge (Mathematical Interpretation):
Unlike standard black-box implementations, this project focuses on:
Statistical Correlation: Using Pearson Correlation Coefficient to filter redundant features.
Residual Analysis: Evaluating the model using $R^2$ and $MSE$ (Mean Squared Error) to understand the deviation from the mean.
Entropy & Information Gain: Explaining how the Decision Tree splits data based on minimizing impurity (Mathematical Entropy).
